# こたつWorldハウスって？

RubyWorld Conferenceでこたつが運営する、RubyKaigiにおけるエモリハウスのオマージュです。
こたつは鳥取県米子市に住んでおり、松江まで往復1hといった感じなんですが、運転があまり好きではないのでいっそ松江に泊まって飲み歩くか～くらいのノリでいます。

## 詳細

key value
期間 11/9-11/12（前泊、後帰り）
場所 ゲストハウス小村 link
部屋数 共用スペース + 寝室3部屋
人数 2022/09/18時点で2人確定、気になる組が2人
上限 7人ですが、5人までのところで締め切る予定です
2022/09/18時点で3人の予約を取っています

## Q&A

  • 領収書は人数分出ますか？
    • システム手数料を除いた人数分頭割りで出せます
    • 9日は3人、10日は4人と言ったかたちで日ごとの滞在人数が変更の場合は、日ごとの人数分布を無視し、頭割りになります
  • 予算感について
    • 一泊22,000＋システム利用料
    • 3人なら7千円ちょい、4人なら6千円弱って感じ
    • 5人だと1泊当たりの価格がかわるので5千円見ておけばよさそうです
